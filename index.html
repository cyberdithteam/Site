Instalação Recursos Como Usar Ferramentas GitHub

Instalação no Termux

Siga estes passos para instalar o FSociety Toolkit no Termux:

pkg update -y && pkg upgrade -y
pkg install -y git wget python
wget https://raw.githubusercontent.com/cyberdithteam/fsociety-date/main/fsociety.py
chmod +x fsociety.py
pip install requests pkg install exiftool python fsociety.py

Para instalação completa com todas as ferramentas:

python fsociety.py --install-all

Como Usar

Menu Principal

O toolkit oferece um sistema de menu numérico intuitivo:

1. Scanner de Rede
2. Ferramentas de Análise
3. Ferramentas OSINT
4. Testes de Segurança
5. Configurações
6. Sair

Exemplo: Verificação de Portas

1. Selecione "Scanner de Rede"
2. Escolha "Verificação de Portas"
3. Digite o endereço IP
4. Aguarde os resultados
